# 🐍 Desafios Python Iniciantes

Este repositório contém uma coleção de **5 desafios práticos em Python** desenvolvidos para praticar lógica de programação, operações matemáticas, manipulação de listas, estruturas de controle e validação de entrada. Todos os exercícios são executados via terminal.

---

## 📚 Exercícios

### 1. Calculadora Simples
Permite realizar operações básicas:

- Adição (`+`)
- Subtração (`-`)
- Multiplicação (`*`)
- Divisão (`/`)
- Tratamento de divisão por zero e entrada inválida

**Conceitos aplicados:**
- `input()`
- `if/elif/else`
- `try/except`
- Operadores matemáticos

---

### 2. Gerador de Tabuada
Gera a tabuada de um número informado pelo usuário.

**Conceitos aplicados:**
- Laço `for`
- Validação de entrada
- Conversão de tipos (`int`)
- Estrutura de repetição

---

### 3. Lista de Números e Estatísticas
Recebe 5 números e exibe:

- Lista completa
- Maior número
- Menor número
- Média dos valores

**Conceitos aplicados:**
- Listas
- `append()`
- `max()` e `min()`
- Laços de repetição
- Tratamento de erro

---

### 4. Sistema de Média Escolar
Calcula a média de três notas e informa a situação do aluno:

- **Aprovado** → média ≥ 7  
- **Recuperação** → média entre 5 e 6.9  
- **Reprovado** → média < 5  

**Conceitos aplicados:**
- Operadores matemáticos
- Estruturas condicionais `if/elif/else`
- Validação de entrada com `try/except`
- Média aritmética

---

### 5. Verificador de Número Par ou Ímpar
Verifica se um número é par ou ímpar.

**Conceitos aplicados:**
- Operador módulo `%`
- Estruturas condicionais
- Validação de entrada

---

## 🛠 Tecnologias Utilizadas

- Python 3
- Terminal / Console
- Conceitos de lógica de programação, listas, operadores, condicionais e loops

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/desafios-python-iniciantes.git
```

2. Acesse a pasta:

```bash
cd desafios-python-iniciantes
```

3. Execute o arquivo desejado:

```bash
python nome_do_arquivo.py
```
