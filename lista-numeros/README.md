# Lista de Números e Estatísticas

## Objetivo

Criar um programa em Python que receba números do usuário e calcule estatísticas básicas.

## Descrição

O programa deve:

1. Solicitar 5 números ao usuário.

2. Armazenar os números em uma lista.

3. Exibir a lista completa.

Mostrar:

1. O maior número

2. O menor número

3. A média dos valores

## Exemplo de Execução

Digite o número 1: 10
Digite o número 2: 5
Digite o número 3: 8
Digite o número 4: 2
Digite o número 5: 15

Lista: [10, 5, 8, 2, 15]
Maior número: 15
Menor número: 2
Média: 8.0
