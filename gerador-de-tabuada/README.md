# Gerador de Tabuada

## Objetivo

Criar um programa em Python que gere a tabuada de um número informado pelo usuário.

## Descrição

O programa deve:

1. Solicitar um número inteiro ao usuário.

2. Validar a entrada, garantindo que seja um número válido.

3. Exibir a tabuada do número digitado.

4. Mostrar as multiplicações organizadas em formato claro.

## Exemplo de Execução

Digite um número: 5
5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
