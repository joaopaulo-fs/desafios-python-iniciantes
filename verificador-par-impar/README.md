# Verificador de Número Par ou Ímpar

## Objetivo

Criar um programa em Python que verifique se um número inteiro é par ou ímpar.

## Descrição

O programa deve:

1. Solicitar um número inteiro ao usuário.

2. Validar a entrada, garantindo que seja um número inteiro válido.

3. Verificar se o número é divisível por 2.

4. Exibir se o número é par ou ímpar.

## Exemplo de Execução

Digite um número inteiro: abc
Entrada inválida! Digite apenas números inteiros.
Digite um número inteiro: 10
O número 10 é par.
