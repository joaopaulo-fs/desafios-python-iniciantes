# Sistema de Média Escolar

## Objetivo

Criar um programa em Python que calcule a média de três notas e informe a situação do aluno.

## Descrição

O programa deve:

1. Solicitar três notas ao usuário.

2. Validar as entradas, garantindo que sejam números válidos.

3. Calcular a média aritmética das notas.

4. Informar a situação do aluno de acordo com a média:

Aprovado → média maior ou igual a 7

Recuperação → média entre 5 e 6.9

Reprovado → média menor que 5

## Exemplo de Execução

Digite a nota 1: 8
Digite a nota 2: 6
Digite a nota 3: 7
A média é 7.0 e o aluno está aprovado!
