# Calculadora Simples

## Objetivo
Criar uma calculadora em Python que permita ao usuário realizar operações básicas: adição, subtração, multiplicação e divisão.

## Descrição
O programa deve:
1. Solicitar dois números ao usuário.
2. Perguntar qual operação ele deseja realizar (`+`, `-`, `*`, `/`).
3. Exibir o resultado da operação.
4. Tratar erros como divisão por zero ou entrada inválida.

## Exemplo de Execução

Digite o primeiro número: 10
Digite o segundo número: 5
Escolha a operação (+, -, *, /): /
Resultado: 2.0
